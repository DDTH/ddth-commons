## Installation #

Latest release version: `0.1.2`. See [RELEASE-NOTES.md](RELEASE-NOTES.md).

Maven dependency:

```xml
<dependency>
	<groupId>com.github.ddth</groupId>
	<artifactId>ddth-commons</artifactId>
	<version>0.1.2</version>
</dependency>
```
